# Google-logo
This Python project uses the built-in turtle graphics library to draw the Google logo programmatically.
The script creates a turtle object, adjusts its drawing properties (color, pen size, and speed), and then carefully combines arcs, circles, straight lines, and filled regions to recreate the colorful “G” logo using official Google brand colors.
It is a beginner-to-intermediate level graphics project that demonstrates how geometric shapes and arc segments can be combined to form complex designs.
